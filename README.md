# Image Filter

This is a web app that you can use to add filters to your image, powered by OpenCV functions:
- Brightness (`cv2.convertScaleAbs()`)
- Blur (`cv2.GaussianBlur()`)
- Enhance (`cv2.detailEnhance`)
